Tentang Proyek

UKDC Virtual Campus Tour adalah aplikasi tur kampus interaktif 3D yang dibangun dengan Unreal Engine 5 sebagai proyek akhir S1 Ilmu Informatika Universitas Katolik Darma Cendika (UKDC) Surabaya. Pengunjung dapat menjelajahi gedung kampus, berinteraksi dengan NPC yang merepresentasikan staf dan dosen, serta mengenal fasilitas akademik secara imersif — tanpa harus datang ke kampus secara langsung.

Fitur

🎮 Third-person character controller dengan animasi idle, jalan, dan lari
💬 Sistem dialog NPC gaya RPG dengan pilihan topik percakapan
🛗 Elevator fungsional antar lantai dengan efek fade dan animasi pintu
🧑‍🎓 Pilih karakter (pria/wanita) dan masukkan nama pemain
🏛️ Eksplorasi gedung kampus UKDC secara bebas
🖥️ Standalone — tidak perlu install Unreal Engine

Cara Install & Main

Download file ZIP di atas
Extract ke folder mana saja (pastikan ada ruang kosong ~3 GB)
Buka folder hasil extract → jalankan UKDCTourCampus.exe
Pilih karakter, masukkan nama, dan mulai eksplorasi!
